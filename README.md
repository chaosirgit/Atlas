## 初始目标
1. 可以自己写工具 `tool` 并持久化代码.更新自己
2. 默认已有的工具是网络访问.和搜索,这样在以后思考时可以访问各个能力或(tool)的官方文档.
3. 自我迭代使用 git 版本控制,用中文记录好每次的更新描述.
4. 后续更加完善的功能我会让它自己迭代出来.
5. 迭代过程中如果需要安装python库.则在命令行运行 uv add ...
6. 当然别忘了记忆系统.

## 意图
我本意是想做一个超级智能体.这个智能体可以自我更新一些能力..比如.我说 Atlas .看下明天天气怎么样?
然后他的思考逻辑是要分步骤的: 
1. 先看看它本身有没有获取天气的 tool
2. 如果没有,就进入为自己新增获取天气的能力
3. 新增能力的逻辑是: 先使用搜索能力看下哪个天气接口适合它,或者用户直接指明哪个平台的接口.
4. 然后启动阅读网页能力查看该平台的文档,然后看看是否需要新增python 库,如果需要,就使用命令行执行工具执行安装python库(uv add...) 检查安装完成
5. 开始调用创建文件和写入文件的工具写入获取天气的工具. 期间如果需要APIKEY.就默认在.env 里配置. 如果各种依赖工具没有则会写依赖工具...
所以.这个并不是一个工具助手.它是一个超级智能体!支持自我更新迭代的智能体!.这种分析步骤我只是描述了大概.你需要深刻的理解,然后更好的完成它.
